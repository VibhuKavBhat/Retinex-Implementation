# Retinex-Implementation

## Retinex
Retinex theory (coined from the term Retina + Cortex) states that an image can be seperated into two main parts; its reflectance and its illuminance. Here, we try to enhance our image quality by removing this illuminance factor by various different techniques. This repository implements some of the many methods, and by doing so, we aim to do a comparative study on the different methods.
